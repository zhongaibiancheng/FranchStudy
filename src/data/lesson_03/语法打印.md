# 第三课 法语语法

## **L’imparfait de l’indicatif** + 外来名词复数 + 课文动词变位

---

## 1. 一句话核心

**Imparfait = 过去的“背景 / 状态 / 习惯 / 正在进行”。**
用来讲：**“那时候一直是这样。”**

---

## 2. 什么时候用？（5 大应用场景）

### A. 外貌 / 穿着（过去的“样子”）

* **M. Andrieu était grand et mince.**
  安德里厄先生当时又高又瘦。
* **Il avait des lunettes de soleil.**
  他当时戴着太阳镜。

### B. 心理 / 身体状态

* **Julie était très malade.**
  朱莉当时病得很重。
* **J’avais mal à la tête.**
  我当时头疼。

### C. 过去长期情况 / 习惯

* **Avant, ils habitaient la banlieue nord.**
  以前他们住在北郊。
* **Quand il était petit, il passait ses vacances chez ses grands-parents.**
  他小时候常在祖父母家过假期。
* （课文）**Nous habitions dans la banlieue nord.**
  我们当时住在北郊。

### D. 环境 / 气氛（故事背景）

* **Il faisait nuit… la rue était déserte.**
  天黑了……街上空荡荡的。
* **Ce jour-là, il faisait beau, le ciel était bleu et le soleil brillait.**
  那天阳光很好，天空湛蓝，太阳照耀。

### E. 同时发生 / 一个动作在进行中

* **À neuf heures, son mari est rentré, elle lisait.**
  九点她丈夫回来了，她当时正在读书。
* **Le professeur est arrivé pendant qu’il pleuvait.**
  老师到的时候，雨还在下。
* **Elle parlait quand son mari est entré.**
  她正说话时，她丈夫进来了。

---

## 3. 怎么变位？（最省力公式）

### ✅ 公式

**“nous 现在时”去掉 -ons + 加 imparfait 词尾**

### ✅ 词尾表（必须背）


| 人称        | 词尾   |
| ----------- | ------ |
| je          | -ais   |
| tu          | -ais   |
| il / elle   | -ait   |
| nous        | -ions  |
| vous        | -iez   |
| ils / elles | -aient |

### ✅ 课本常见动词速记

* **parler** → je parlais / nous parlions / ils parlaient
* **finir** → je finissais / nous finissions / ils finissaient
* **avoir** → j’avais / nous avions / ils avaient
* **être** → j’étais / nous étions / ils étaient

> 看到也别怕（课本例）：
> **commencer → je commençais**
> **manger → nous mangions**
> **étudier → nous étudiions / vous étudiiez**

## 4. Imparfait vs Passé composé（必考对比）

### ✅ 记忆图像

**Imparfait = 背景镜头**
**Passé composé = 关键动作镜头**


| 你想表达…               | 用什么？                        | 课本例句（含中文）                                                               |
| ------------------------ | ------------------------------- | -------------------------------------------------------------------------------- |
| 过去的背景/状态/习惯     | **Imparfait**                   | **Avant, ils habitaient la banlieue nord.**以前他们住在北郊。                    |
| 正在进行 vs 突发事件     | **Imparfait + Passé composé** | **À neuf heures, son mari est rentré, elle lisait.**九点他回来了，她正在读书。 |
| 天气/环境背景 + 事件发生 | **Imparfait + Passé composé** | **Le professeur est arrivé pendant qu’il pleuvait.**老师到时正在下雨。         |
| 一个动作“发生并完成”   | **Passé composé**             | **J’ai glissé et je suis tombée.**我滑倒并摔倒了。                            |
| “当时一直有……”       | **Imparfait**                   | **Il y avait de la neige…**当时有雪……                                         |

## 5. 30 秒口诀（背这个就能做题）

**“外貌状态用未完，**
**天气背景也未完；**
**以前习惯常用它，**
**正在进行是未完；**
**突然发生用复合，**
**两者一起最常见。”**

---

## 6. 外来名词的复数（初二超简版）

### ✅ 最重要规则

**不确定怎么变复数时，绝大多数情况“直接加 s”。**

### ✅ 课本给的常见例子

* **une interview → des interviews**
  采访 → 多个采访
* **un forum → des forums**
  论坛 → 多个论坛
* **un scénario → des scénarios**
  剧本/方案 → 多个剧本/方案
* **un visa → des visas**
  签证 → 多个签证

### ✅ 少数“特殊记忆型”（看见就记）

* **un gentleman → des gentlemen**
  绅士 → 绅士们
* **un policeman → des policemen**
  警察 → 警察们
* **un condottiere → des condottieri**
  雇佣兵 → 雇佣兵们
* **un duplicata → des duplicata(s)**
  副本 → 多份副本

> **初二策略**：
> 只要记住一句话：
> **“大多数加 s，少数当成固定词组记。”**

## 7. 课文高频动词变位（用来读课文不卡）

这三个动词是书后“动词变位”专门为课文服务的。

### 7.1 grossir（变胖）

* **Présent**：je grossis / nous grossissons / ils grossissent
* **Futur**：je grossirai / nous grossirons
* **Participe passé**：**grossi**
* （课文）**Il a certainement grossi ces dernières années.**
  他这些年肯定胖了。

### 7.2 nettoyer（打扫）

* **Présent**：je nettoie / nous nettoyons / ils nettoient
* **Futur**：je nettoierai / nous nettoyerons
* **Participe passé**：**nettoyé**
* （课文场景句式）**Ensuite, elle nettoie les escaliers.**
  然后她打扫楼梯。

### 7.3 souffrir（受苦/忍受）

* **Présent**：je souffre / nous souffrons / ils souffrent
* **Futur**：je souffrirai / nous souffrirons
* **Participe passé**：**souffert**
* （课文）**Alors, en attendant, il faut bien souffrir un peu.**
  所以在那之前，只好先忍一忍/吃点苦。
