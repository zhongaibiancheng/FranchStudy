
# 简单将来时 *le futur simple*（初二简化版讲义）

> **一句话记住：**  
> 现在说：**将来某个时候会发生的事** → 用 **简单将来时**。

---

## 一、简单将来时是干嘛用的？

- 表示以后 **一定 / 很可能会发生的事情**  
  - *Demain, nous irons à la campagne.*  
    → 明天我们会去乡下。
- 表示对未来的 **计划、预测、承诺**  
  - *Je serai médecin.*  
    → 我将来会当医生。  
  - *Je vous appellerai demain.*  
    → 我明天会给您打电话。

---

## 二、怎么“造”简单将来时？（最重要）

### 1. 所有动词共同的词尾

在词干后加这一套词尾（一定要背）：

- **je** ：-ai  
- **tu** ：-as  
- **il / elle** ：-a  
- **nous** ：-ons  
- **vous** ：-ez  
- **ils / elles** ：-ont  

---

### 2. 规则动词三种情况

1. **-er 动词**：直接用不定式 + 词尾  
   - *parler* → je parler**ai**, nous parler**ons**

2. **-ir 动词**：也是不定式 + 词尾  
   - *finir* → tu finir**as**, vous finir**ez**

3. **多数 -re 动词**：先去掉最后一个 -e，再加词尾  
   - *attendre* → j’attendr**ai**, nous attendr**ons**

> 小提醒：发音时 *je parlerai / tu parleras / il parlera* 很像，  
> 真正区分要看“主语 + 词尾”。

---

## 三、最常见“不老实动词”的将来时词干

这些动词的**词干要单独记**，但后面跟的词尾仍然是那一套：-ai, -as, -a, -ons, -ez, -ont。

| 不定式   | 将来时词干 | 例子（je / nous）                              |
|----------|------------|-----------------------------------------------|
| aller    | ir-        | j’**irai**, nous **irons**                     |
| avoir    | aur-       | j’**aurai**, nous **aurons**                   |
| être     | ser-       | je **serai**, nous **serons**                  |
| faire    | fer-       | je **ferai**, nous **ferons**                  |
| venir    | viendr-    | je **viendrai**, nous **viendrons**            |
| voir     | verr-      | je **verrai**, nous **verrons**                |
| vouloir  | voudr-     | je **voudrai**, nous **voudrons**              |
| pouvoir  | pourr-     | je **pourrai**, nous **pourrons**              |
| devoir   | devr-      | je **devrai**, nous **devrons**                |
| savoir   | saur-      | je **saurai**, nous **saurons**                |

> 建议：先把 **je 形 + nous 形** 背熟，其它人称只要会用同一套词尾就能自己推出来。

---

## 四、简单将来时常见用法小结

1. **普通未来**  
   - *Demain, nous irons au cinéma.*  
     → 明天我们会去看电影。

2. **承诺 / 约定**  
   - *Je te téléphonerai ce soir.*  
     → 今晚我会给你打电话。

3. **礼貌的“命令 / 提醒”**（多用 vous）  
   - *Vous fermerez la porte, s’il vous plaît.*  
     → 请您把门关上。（公告、正式语气中常见）
