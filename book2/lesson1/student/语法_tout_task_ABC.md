# TOUT 一词学习清单（A / B / C 分级）

> 核心记一句话：  
> **tout / toute / tous / toutes** 大致都带有  
> “全、都、整个、每、很” 这样的意思。

| 阳单 | 阴单  | 阳复 | 阴复   |
|------|-------|------|--------|
| tout | toute | tous | toutes |

---

## A 级任务：这一课必须会的

> 目标：能看懂、能自己造简单句子，不被 tout 吓到。

### A1. “全 / 每 / 整个” —— 修饰名词（形容词用法）

结构：**tout + 名词**（中间可能有冠词：tout le / toute la / tous les / toutes les）

- 阳单：**tout le jour** 一整天  
- 阴单：**toute la journée** 整个一天  
- 阳复：**tous les jours** 每一天  
- 阴复：**toutes les semaines** 每个星期  

**需要掌握：**

- [ ] 会根据名词，选对形式：

  1. ___ le jour  → **tout**  
  2. ___ la journée → **toute**  
  3. ___ les élèves  → **tous**（所有学生）  
  4. ___ les filles   → **toutes**（所有女生）

- [ ] 能看懂并最好背下来两句：

  - *Il a dormi **toute la journée***.  
    他睡了一整天。  

  - *On y va **tous les ans***.  
    我们每年都去那儿。

---

### A2. “全部 / 一切 / 都” —— 不带名词（代词用法）

不用名词、直接用 **tout / tous**，表示“全部、一切、所有人”。

- *Tout va bien.* 一切都好。  
- *Je comprends tout.* 我全都懂。  
- *Ils sont tous là.* 他们都在这儿。

**需要掌握：**

- [ ] 看中文能写出含 tout / tous 的法语句子：

  1. 一切都好。→ **Tout va bien.**  
  2. 我全都懂了。→ Je comprends **tout**.  
  3. 他们都在这儿。→ Ils sont **tous** là.

---

### A3. “很、非常” —— 修饰形容词（副词用法，最简单版）

相当于 very，只用最基本两句：

- *Il est **tout** content.*  
  他很高兴。（男的）

- *Elle est **toute** contente.*  
  她很高兴。（女的）

**需要掌握：**

- [ ] 能根据主语性别，选 tout / toute：

  1. Paul est ___ content. → **tout**  
  2. Marie est ___ contente. → **toute**

---

### ✔ A 级小结（自检）

- [ ] 我知道 tout 系列大概表示 “全 / 都 / 整个 / 每 / 很”。  
- [ ] 我会用 tout / toute / tous / toutes + 名词 说“整天、每年、所有人……”。  
- [ ] 我会用 tout / tous 表示 “一切 / 都”。  
- [ ] 我会说：*Il est tout content.* / *Elle est toute contente.*  

以上都能打勾 = 本课关于 **tout 的基础 OK ✅**。

## B 级任务：巩固 + 和课文结合（有精力再做）

> 目标：用 tout 更自然，句子更多一点。

### B1. 在课文里“找 tout”

- [ ] 拿出课文，用彩笔圈出所有 **tout / toute / tous / toutes**。  
- [ ] 每圈到一个，在旁边标注一下中文大意，比如：  
  - tout le monde → 所有人  
  - tout cela → 这一切  
  - toutes les vacances → 整个假期  

### B2. 造句练习（综合运用）

用下列提示，自己造句（每种至少 1 句）：

1. **形容词用法（+ 名词）**  
   - 提示：toute la soirée（整个晚上）、tous les élèves、toutes les vacances  
   - 例：*J’ai travaillé **toute la soirée**.*  

2. **代词用法（= 一切 / 都）**  
   - 提示：tout va… / je fais tout / ils ont tous…  
   - 例：*Nous avons tout vu.*  

3. **副词用法（= 很）**  
   - 提示：tout triste, toute heureuse  
   - 例：*Il est tout triste aujourd’hui.*  

- [ ] 写完以后，读给家长听，让家长能大致听懂意思。

## C 级任务：了解即可，不必现在死记

> 给以后学更高级法语打个“预告”，现在只要知道有这回事。

- [ ] 知道在 **否定句** 里，有时是否定 “全部”：  
  - *Tout le monde n’est pas d’accord.*  
    → 不是所有人都同意。（不是每个人都同意）

- [ ] 知道有一些 **固定短语** 里有 tout：  
  - *tout le monde*（所有人）  
  - *pas du tout*（一点也不）  
  - *tout de suite*（马上）  
  现在只要见到能认出来，大致懂意思即可。

- [ ] 知道当 **tout** 做副词修饰阴性形容词时，  
      有时会变成 **toute / toutes**，比如：  
  - *Elle est toute contente.*  
  - *Elles sont toutes contentes.*  
  以后有时间再系统总结，现在只记几个例句足够。
