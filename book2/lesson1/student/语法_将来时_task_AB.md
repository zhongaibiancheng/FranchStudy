# 将来时 *le futur simple* 学习清单（A + B）

> 给某位初二小朋友的小任务本：
> 先完成 **A 级任务**，再慢慢挑战 **B 级任务**。
> 每做完一条，打一个 ✅。

---

## 一、A级任务：基础必修清单

### 1. 概念 & 用法（先知道在干嘛）

- [ ]  我能用自己的话解释：
  **“将来时就是——现在说：以后会发生的事。”**
- [ ]  我知道将来时最常用的两种情况：

  1. **普通未来**

     - 表示以后会发生的事情
     - 例：
       - Demain, nous **irons** au cinéma.
         明天，我们将去电影院。
       - Je **serai** médecin.
         我将成为医生。/ 我将来要当医生。
  2. **如果……就……（现实条件句）**

     - 结构：**Si + 现在时，主句用将来时**
     - 例：
       - S’il fait beau demain, nous **irons** à la campagne.
         如果明天天气好，我们就去乡下。
       - Si j’ai le temps, je **ferai** mes devoirs.
         如果我有时间，我就做作业。

---

### 2. 词尾 & 规则动词（将来时长什么样）

#### 2.1 背将来时 6 个词尾

- [ ]  我能在纸上写出：
  je **-ai** 我
  tu **-as** 你
  il/elle **-a** 她/他
  nous **-ons** 我们
  vous **-ez** 你们/您们/您
  ils/elles **-ont** 他们/她们
- [ ]  老妈 / 老爸遮住书本，我还能默写出来 ✅

#### 2.2 会用规则动词变将来时

以 **parler（说）** 为例：

- [ ]  我能写出完整六个形式：
  je parlerai
  tu parleras
  il/elle parlera
  nous parlerons
  vous parlerez
  ils/elles parleront

再用一个 **-ir 动词**（finir 或 choisir）练习：

- [ ]  我能写出：
  je finirai / choisirai
  nous finirons / choisirons
  ils finiront / choisiront

> 小口诀：
> 大部分 **-er, -ir** 动词：
> **不定式 + 将来时词尾** 就搞定。

### 3. 10 个常用不规则动词（只要 je + nous）

> 目标：看中文能写出 **je 的将来时**，最好也能说出 **nous 形式**。

- [ ]  我抄过下面这张表一次，并读过一遍：


| 动词          | je（我将会……） | nous（我们将会……） |
| ------------- | ---------------- | -------------------- |
| aller(去)     | j’irai          | nous irons           |
| avoir(有)     | j’aurai         | nous aurons          |
| être(是)     | je serai         | nous serons          |
| faire(做)     | je ferai         | nous ferons          |
| venir(来)     | je viendrai      | nous viendrons       |
| voir(看)      | je verrai        | nous verrons         |
| vouloir(想要) | je voudrai       | nous voudrons        |
| pouvoir(能)   | je pourrai       | nous pourrons        |
| devoir(必须)  | je devrai        | nous devrons         |
| savoir(知道)  | je saurai        | nous saurons         |

- [ ]  老妈 / 老爸说中文：“去、是、有、做、来、看、想要、能、必须、知道”，
  我能写出对应的 **je 将来时**（比如：j’irai, je serai…）。

---

### 4. 造句练习（确保能用出来）

#### 4.1 普通将来句

- [ ]  我自己写出至少 **3 句**“以后会怎么样”的句子，例如：

  - Demain, je **ferai** mes devoirs.
    明天，我将做我的作业。
  - La semaine prochaine, nous **irons** au cinéma.
    下周，我们将去看电影。
  - Plus tard, je **vivrai** à l’étranger.
    以后，我将住在国外。

（写好后读给家长听一遍 ✅）

#### 4.2 “如果……就……”句型（si + 现在时，主句将来）

- [ ]  我自己写出 **2 句**“si + 现在时，主句将来”的句子，例如：

  - Si j’ai le temps, je **jouerai** au foot.
    如果我有时间，我就去踢足球。
  - S’il fait beau demain, nous **irons** faire du ski.
    如果明天天气好，我们就去滑雪。

A级任务全部打勾 ✅ =
**说明：我已经会用将来时说简单的未来计划和“如果……就……”了。**

## 二、B级任务：提升巩固清单

> 在 A 级基础上，多走一步，让将来时“更熟、更自然”。

### 1. 和课文结合（在书上动笔）

- [ ]  在课文里，用彩笔 **圈出所有将来时动词**（比如 *irons, découvrirez, ferez…*）。
- [ ]  对每一句含将来时的句子，用中文在旁边写一句大意。
  （大意对就行，不要追求逐字翻译）

例子：

- Vous découvrirez la Bretagne en vélo.
  → 你们会骑车游览布列塔尼。

---

### 2. 句型转换（现在时 ↔ 将来时）

- [ ]  把老师给的 / 课本上的 **2～3 句现在时**，改写成将来时，例如：

  - Aujourd’hui, nous allons au cinéma.
    → Demain, nous **irons** au cinéma.
  - Aujourd’hui, je fais mes devoirs.
    → Ce soir, je **ferai** mes devoirs.
- [ ]  反过来：把自己写的 1～2 句将来时，改成现在时，检查自己有没有真正看懂。

---

### 3. 小短文（3～5 句）

任选一个主题写一小段话：

- 题目 1：**Demain, je…**（明天我会做什么）
- 题目 2：**Plus tard, je serai…**（将来我会成为什么）

要求：

- [ ]  至少写 **3～5 句**，里面至少用 **3 次将来时**（可以重复同一个动词）。
- [ ]  写完后，大声读一遍给家长听，让家长大概能听懂中文意思。

---

### 4. 拼写小变化（只管课文里遇到的几个）

只要求对这类变化“有印象”，看见不害怕：

- acheter → j’achèterai
- se lever → je me lèverai
- essayer → j’essaierai
- envoyer → j’enverrai

- [ ]  我能看懂上面这几句的意思，并知道：
  “哦，这些是为了发音好听，所以拼写有一点点变。”

> 不要求全部背，只要课文里遇到的那几个看见能认出来就好。
