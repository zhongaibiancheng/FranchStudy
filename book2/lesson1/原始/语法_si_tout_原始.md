# Leçon 7 语法总结

## 2. Si 引导的假设从句（现在时）

### (1) 概念 (Notion)

如果从句事情的实现虽非主句事情的实现所必需的条件，但已足够保证主句事情的实现，则从句不构成条件，而构成假设。这类假设句表示假设的成分较多，且实现的可能性较大。

*注：假设和条件常常紧密联系或混合在一起。多数语法讲解中并未将假设从句和条件从句加以细分，而是将二者合称为条件句。*

### (2) 构成 (Formation)

**【结构】**：`si` + 从句（动词使用**直陈式现在时**），主句（动词使用**现在时**、**将来时**或**命令式**）。

**核心规则**：假设从句的动词**只用直陈式现在时，不能使用简单将来时！** 而主句动词可视情况使用现在时、将来时或命令式。

### (3) 用法 (Emploi)

| 法语例句 | 中文翻译 |
| :--- | :--- |
| On va au cinéma si tu es libre ce soir. | 要是你今晚有空儿，咱们去看电影。 |
| Si vous lui pardonnez, vous êtes généreux. | 假如您能宽恕他，那您就是宽宏大量了。 |
| S'il fait beau demain, nous irons à la Grande Muraille. | 如果明天天气好，我们就去长城。 |
| Si elle continue ainsi, elle réussira tôt ou tard. | 只要一如既往，她迟早会成功。 |
| Si vous êtes d'accord, on commencera tout de suite. | 若是您同意，咱们马上开始。 |
| Si tu passes par le magasin, achète-moi un pain. | 你要是路过商店，给我买个面包。 |
| Jean, laisse-lui un mot s'il n'est pas là! | 让，如果他不在，给他留个字条儿。 |

## 3. Tout, Toute, Tous, Toutes 的用法 (L‘emploi de…)

**词性**：`tout`/`toute`/`tous`/`toutes` 可以是**泛指形容词**、**泛指代词**或是**副词**，具体词性应视其在句中所起的语法作用而定。

### 1) 作泛指形容词 (L’adjectif indéfini)

置于冠词、主有形容词、指示形容词、指示代词或无冠词名词前时，是泛指形容词。此时，其**性数必须与其所限定的名词的性数保持一致**。

#### 用法例句

| 法语例句 | 中文翻译 | 备注 |
| :--- | :--- | :--- |
| Il a dormi **toute la** journée. | 他睡了整整一天。 | 修饰阴性单数名词 |
| Elle a lu **toute une** nuit. | 她读了一整夜的书。 | 修饰阴性单数名词 |
| J‘ai vu **toute sa** famille. | 我见到了他的全家。 | 修饰阴性单数名词 |
| **Tous les** chemins mènent à Rome. | 殊途同归。（条条大路通罗马。） | 修饰阳性复数名词 |
| On y va **tous les deux** ans. | 我们每两年去那儿一次。 | 修饰阳性复数名词 |
| Elles sont là **toutes les trois**. | 她们三个都在。 | 修饰阴性复数名词 |
| Il a fait très chaud **tout cet** été. | 今年整个夏天都很热。 | 修饰阳性单数名词 |
| **Tout ceci** est peu important. | 这一切都不重要。 | 修饰指示代词 |
| **Tout cela** est difficile à comprendre. | 这一切都太难懂了。 | 修饰指示代词 |
| **Tout ça** a l‘air délicieux. | 这些看起来都很好吃。 | 修饰指示代词 |
| **Tout homme** peut se tromper. | 任何人都会出错。 | 修饰无冠词名词 |
| Je vous remercie de **tout cœur**. | 我由衷地感谢您。 | 固定表达 |
| **De toute façon**, elle ne viendra pas. | 无论如何她都不会来了。 | 固定表达 |

#### 在否定句中的用法

在否定句中，被否定的不是动词，而是泛指形容词 `tout`/`toute`/`tous`/`toutes`。

| 法语例句 | 中文翻译 |
| :--- | :--- |
| **Tout le monde** n‘est **pas** descendu. | **不是所有人**都下楼了。 |
| **Toute la** classe n‘est **pas** d’accord. | **不是全班**都同意。 |
| **Tous les** étudiants ne sont **pas** rentrés. | **不是全体**学生都返校了。 |
