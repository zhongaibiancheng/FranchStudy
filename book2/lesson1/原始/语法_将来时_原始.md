# GRAMMAIRE 语法

## 1. Le futur simple de l’indicatif（直陈式简单将来时）

### 1）概念（Notion）

简单将来时是相对现在（说话）时刻而言，表示确定或可能在将来、未来、今后发生的动作。

### 2）构成（Formation）

在法语中，除少数第三组不规则动词外，大部分动词的简单将来时基本都是以动词不定式为词干构成，其变位都是规则的。  

※ 所有动词的简单将来时均由字母 **r** 加词尾构成，即：  
**-rai, -ras, -ra, -rons, -rez, -ront**

请看下列表：

#### 表 1：简单将来时构成表

| 动词类别   | 动词不定式                 | 变位规则                      | 动词变位示例                                  |
| ---------- | -------------------------- | ----------------------------- | --------------------------------------------- |
| 第一组动词 | parler                     | 直接加词尾（少数动词变位特殊，见表 2） | je parlerai / nous parlerons                  |
| 第二组动词 | finir                      | 直接加词尾                    | tu finiras / vous finirez                     |
| 第三组动词 | mettre, prendre 等以 -re 结尾 | 先去掉 **-e**，再加词尾        | je mettrai / nous mettrons ; il prendra / ils prendront |
| 第三组动词 | sortir, ouvrir 等以 -ir 结尾 | 直接加词尾                    | tu sortiras / vous sortirez ; j’ouvrirai / nous ouvrirons |
| 第三组动词 | devoir, recevoir, pleuvoir 等以 -oir 结尾的少数动词 | 去掉 **-oir**，加 **-r**，再加词尾 | je devrai / nous devrons ; je recevrai / vous recevrez ; il pleuvra |

> *大多数以 -oir 结尾的动词的简单将来时动词变位不规则。*

出于发音的原因，第一组动词中的 **lever / répéter** 以及以 **-eter / -eler / -oyer / -uyer / -ayer** 结尾的动词在直陈式现在时的动词变位必须在其变位中做出相应的变化。这种情况也延伸到其简单将来时的动词变位中。

#### 表 2：第一组动词特殊变位表

| 第一组特殊动词不定式 | 变位规则                                      | 动词变位示例                                   |
| -------------------- | --------------------------------------------- | ---------------------------------------------- |
| jeter                | 双写 **t**，再加词尾                          | je jetterai / nous jetterons                   |
| appeler              | 双写 **l**，再加词尾                          | j’appellerai / nous appellerons                |
| acheter              | 将 **e** 改为 **è**，再加词尾                 | j’achèterai / nous achèterons                  |
| se lever             | 将 **e** 改为 **è**，再加词尾                 | je me lèverai / nous nous lèverons             |
| répéter              | （与现在时不同）保留 **é**，再加词尾          | je répéterai / nous répéterons                 |
| espérer              | （与现在时不同）保留 **é**，再加词尾          | j’espérerai / nous espérerons                  |
| essayer              | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | j’essaierai / nous essaierons                  |
| balayer              | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | je balaierai / nous balaierons                 |
| payer                | 1. 改 **y** 为 **i**，再加词尾；2. 直接加词尾 | je paierai / nous paierons；je payerai / nous payerons |
| nettoyer             | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | je nettoierai / nous nettoierons               |
| tutoyer              | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | je tutoierai / nous tutoierons                 |
| appuyer              | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | j’appuierai / nous appuierons                  |
| essuyer              | 为软化发音，将不发音的 **e** 前的 **y** 改为 **i**，再加词尾 | j’essuierai / nous essuierons                  |
| envoyer              | 不规则变位                                   | j’enverrai / nous enverrons                    |
| renvoyer             | 不规则变位                                   | je renverrai / nous renverrons                 |

### 3）用法（Emploi）

简单将来时表示将要在**近期或遥远的将来**发生的动作，或者表示在未来的一段时间内将要存在的状态。  
使用时经常伴有表示时间的状语（时间标志）。

#### ① 表示将来肯定会发生的事情或状态

- De toute façon, je ne resterai pas ici l’an prochain.  
  明年无论如何我也不呆在这儿了。

- Avec l’inflation, les prix augmenteront encore.  
  随着通货膨胀，物价肯定还得涨。

- Les oiseaux reviendront au printemps.  
  鸟儿春天就会回来。

- Qu’est-ce que ce village deviendra dans 10 ans ?  
  这个村庄十年后会变成什么样呢？

- Dans le futur, on pourra voyager dans l’espace.  
  将来人们可以去太空旅行。

#### ② 表示将来可能发生的事情或状态（推测）

- Il pleuvra demain d’après la météo.  
  天气预报说明天（可能）有雨。

- Si tout va bien, elle y arrivera dans 24 heures.  
  如果一切顺利，24 小时后她就能到那儿。

- Le directeur est absent, il sera malade.  
  主任今天没来，他大概病了。

- — Qu’est-ce que tu feras quand tu seras plus grand ?  
  长大以后你要做什么？  
  — Je serai professeur ou… entraîneur !  
  我要当老师或是……教练！

#### ③ 礼貌性将来时（Le futur de politesse）

简单将来时此时用作第一人称，代替现在时，表示委婉语气。这种在社交场合的用法被称为“礼貌性将来时”。

- Je vous prierai de bien vouloir me suivre.  
  请您随我来。

- Je me permettrai de vous faire remarquer que le train en provenance de Lausanne entrera en gare avec une heure de retard.  
  我冒昧地提醒您注意：由洛桑开来的列车将晚点一小时进站。

#### ④ 表示委婉命令、劝告（代替第二人称命令式）

简单将来时用于第二人称时，具有命令式的意义，表示委婉的愿望、劝告或命令，可代替第二人称命令式。这种用法常见于口语表达中。

- Tu écriras cela 20 fois.  
  你把这句写 20 遍。

- Vous prendrez la première rue à droite.  
  您走右边第一条街。

- Tu ne recommenceras plus !  
  以后别再这样了！

- Vous rappellerez demain matin.  
  您明天上午再来电话吧。
